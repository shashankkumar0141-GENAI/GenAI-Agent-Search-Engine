# GEN AI Search Engine using Agents & Tools

This project demonstrates how to build a **GenAI Search Agent** using LangChain and Groq.

The agent can dynamically choose tools like:

- Arxiv (Research papers)
- Wikipedia
- Web Search

## Tech Stack

- Python
- Streamlit
- LangChain
- Groq LLM
- Arxiv API
- Wikipedia API
- DuckDuckGo Search

## Features

- Tool calling using LangChain Agents
- Web + Research search capability
- Interactive chat interface
- Streaming responses
